# DD
## var dumper for *AdonisJS*

> Like *Laravel dd*

![alt text](./capture.png)

### How to use ?

1. **Installation** (via npm)
```
npm install git@github.com:IgorBressan/dd.git --save
```

2. **usage**
```
var dd = require('houssamdev-dd')
...
  ... dd( yourData, responseObject, [expanded = false] )
```

that's all ^^

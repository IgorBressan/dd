# DD
## var dumper for *NodeJs* (and *ExpressJs*)

> Like *Laravel dd*

![alt text](./capture.png)

### How to use ?

1. **Installation** (via npm)
```
npm install houssamdev-dd
```

2. **usage**
```
var dd = require('houssamdev-dd')
...
  ... dd( yourData, responseObject, [expanded = false] )
```

that's all ^^
